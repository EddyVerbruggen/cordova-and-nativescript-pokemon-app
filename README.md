## Cordova

[Install Cordova](https://cordova.apache.org/docs/en/latest/guide/cli/)

```bash
cd Cordova
cordova platform add ios android
cordova run android|ios
```

## NativeScript

[Install NativeScript](https://docs.nativescript.org/start/quick-setup)

```bash
cd NativeScript
npm i
tns run android|ios
```
